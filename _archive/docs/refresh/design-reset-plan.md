# Design Reset Plan

## Verdict

The current visual pass is not good enough. It feels too much like a generic affiliate template, not a refined Christmas buying guide.

## Keep

- Markdown guide source of truth
- Audit/validation scripts
- Product freshness fields
- AffiliateLink tracking/compliance
- Build/lint/typecheck cleanup
- Refreshed Christmas lights content

## Redo

- Top navigation
- Homepage hero
- Start by Need section
- Guide page layout
- Product cards
- Infographic modules

## Problems to fix

1. Nav text is too small and low-contrast.
2. Guide pages are too text-heavy.
3. Too many stacked blocks before useful content.
4. Visuals are weak or irrelevant.
5. Product cards lack clear “best for” decisions.
6. Page hierarchy feels patched together.
7. Accessibility was not treated as a design requirement.

## New design direction

Plain-language goal:

> Make ChristmasGearHQ feel like a readable, premium holiday buying guide — clear, warm, useful, and easy to scan.

Design rules:

- Larger text.
- Fewer blocks.
- Better contrast.
- More white space.
- Fewer cards.
- Stronger decision modules.
- No decorative visuals unless they help.
- Every section gets one job.

## Rebuild order

1. Navigation
2. Guide page template
3. Product cards
4. Homepage
5. Category pages

## First rebuild target

Start with:

```text
/guides/best-christmas-lights-2026
```

Goal:

- Make it a buying tool, not an article wall.
- Add clear product/decision modules.
- Reduce prose density.
- Keep the useful content.
