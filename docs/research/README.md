## How to use this when building content

1. **Don't fabricate.** Each markdown report is structured research with cited sources. Treat it as the source of truth for product picks, expert-source counts, and editorial framing.
2. **Pillar-first sequencing.** The aquarium hub-and-spoke structure (water-quality hub + filtration hub) is the highest-leverage starting point. Ship the hub pages first, then the spokes — that's how topical authority compounds in SERPs.
3. **Reuse research across spokes.** The cycling/bacteria report cites overlapping authorities with the test-kits report; the same expert sources can anchor both guides.
4. **Bird angle is birdwatching, not pet birds.** If the v2 pitch is "exotic pets" framed broadly, this still fits — bird feeders and backyard-birding gear is a real outdoor/lifestyle vertical with $50-300 AOV and a passionate audience. But mention it as such; don't conflate with parrot/cage-bird content.
5. **Reptile hubs need species-spoke decisions.** Both reptile reports are framed as hubs. Before drafting product guides, decide which species to anchor first (popular: bearded dragon, leopard gecko, ball python, crested gecko). Each species changes the product picks materially.
